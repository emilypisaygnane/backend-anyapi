AnyAPI

This API uses "dogs" and "cars" as its resource. The routes are as follows:
Route Method Description Example 
/dogs
 GET Gets a list of dogs' names and ids List Dogs 
/dogs/:id
 GET Returns the detail about the dog matching the id in the parameter Get Dog 1 
/cars
 GET Gets a list of cars' names and ids List Cars 
/cars/:id
 GET Returns the detail about the car matching the id in the parameter Get Car 1
